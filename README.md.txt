# UniHub Portal 🎓

A centralized college resource portal with features like student performance tracking, course registration, and faculty content management.

---

## 📁 Project Structure

UniHub-Portal/
├── client/ # React frontend
├── server/ # Node/Express backend
├── docs/ # Project documents
└── README.md # This file

---

## 🚀 Features

- Student performance graphs
- Course registration
- Faculty & admin login
- Academic resources (notes, quizzes)

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
